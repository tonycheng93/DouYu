# DouYu

##  Dependencies

> * [RxAndroid](https://github.com/ReactiveX/RxAndroid)
> * [Glide](https://github.com/bumptech/glide)
> * [Retrofit](https://square.github.io/retrofit/)
> * [LeakCanary](https://github.com/square/leakcanary)
